# 秘書室 (Secretary)

RYO AI Company の秘書室です。タスク管理・アイデア記録・思考の整理を行います。

## ファイル構成

| ファイル | 用途 |
|----------|------|
| `inbox.md` | 思いついたことを即メモ。日付付きで追記 |
| `tasks.md` | やること一覧。TODO / DOING / DONE で管理 |
| `ideas.md` | アプリ・事業アイデアの記録 |

## 使い方

### 日々のフロー

1. **思いついたらまず `inbox.md` へ**
   - 細かいことを考えず即書く
   - 日付を先頭に付けて追記

2. **具体的なタスクは `tasks.md` へ**
   - inbox から拾い上げてタスク化
   - ステータスを TODO → DOING → DONE と更新

3. **アイデアは `ideas.md` へ**
   - 事業・アプリのアイデアを記録
   - 優先度・メモを添えて管理

### ルール

- inbox は「まず書く」を優先、後で整理する
- tasks は定期的に DONE を棚卸し
- ideas は腐らせず週1でレビュー
